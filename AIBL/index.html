Count-Preserving Generation in Dense Scenes

Text-to-image diffusion models often merge nearby or occluded objects into visually plausible but count-incorrect outputs. AIBL combines ownership-aware layout attention with an amodal-aware instance-balanced loss to strengthen supervision for partially visible objects and reduce cross-instance feature leakage.

AIBL Amodal-Aware Instance-Balanced Loss for occluded instances.
45K OverlapDepth benchmark scale for dense synthetic layouts.
Layout Soft ownership-aware attention biases for overlapping boxes.

Method Overview

The project focuses on keeping individual object instances separable when prompts and layouts contain many objects, heavy overlap, and large visible-area imbalance.
